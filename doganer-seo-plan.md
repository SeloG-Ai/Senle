## 2. SAYFA OPTİMİZASYONU (On-Page SEO)

### 2.1 Mevcut Sayfalar İçin Öneriler

**Ana Sayfa (index.html)**
```
Title: İstanbul Çekici Hizmeti | 7/24 Araç Çekme | Doğaner Lojistik
Meta: İstanbul'da 7/24 çekici ve yol yardım hizmeti. Net fiyat, 45 dk'da ulaşım. 
      Hemen arayın: 0535 641 03 04
H1: Çekiciye ihtiyacın var. Biz geliyoruz. (✅ zaten var)
```

**Hizmetler Sayfası**
```
Title: Çekici Hizmetleri İstanbul | Araç Çekme, Yol Yardım | Doğaner Lojistik
Meta: İstanbul'da araç çekme, ağır çekici, yol yardım ve kurumsal filo hizmetleri. 
      Şeffaf fiyat, 7/24 destek.
H1: İstanbul Çekici Hizmetleri
```

**İletişim Sayfası**
```
Title: İstanbul Çekici Telefon | 7/24 Ulaşın | Doğaner Lojistik Küçükçekmece
Meta: İstanbul çekici 0535 641 03 04. Halkalı, Küçükçekmece merkezli, 
      tüm İstanbul'a hizmet. 7/24 açık.
```

### 2.2 URL Yapısı

```
doganerlojistik.com/               → Ana Sayfa
doganerlojistik.com/hizmetler/    → Hizmetler
doganerlojistik.com/blog/         → Blog listesi
doganerlojistik.com/blog/[slug]/  → Blog detay
doganerlojistik.com/iletisim/     → İletişim
```

## 5. TEKNİK SEO KONTROL LİSTESİ

### 5.1 Site Hızı (Core Web Vitals)
- [ ] Görselleri WebP formatına dönüştür
- [ ] Görselleri sıkıştır (TinyPNG veya Squoosh)
- [ ] Fontları preload et
- [ ] CSS minify et
- [ ] JS minify et
- [ ] Hosting: Hostinger LiteSpeed Cache aktif mi? Kontrol et
- [ ] Google PageSpeed Insights'ta test et → hedef: 90+ mobile

### 5.2 Mobil Uyum
- [ ] Google Mobile-Friendly Test'ten geç
- [ ] Dokunma alanları 48px minimum
- [ ] Font min 16px
- [ ] Viewport meta tag var (✅ HTML'de mevcut)

### 5.3 Teknik Dosyalar
- [ ] `sitemap.xml` oluştur ve Google Search Console'a ekle
- [ ] `robots.txt` dosyası oluştur
- [ ] Google Search Console'a site ekle ve doğrula
- [ ] SSL sertifikası (HTTPS) aktif mi? Kontrol et

### 5.4 Schema Markup (Yapısal Veri)

Ana sayfaya şu schema'yı ekle — Google'ın zengin sonuç göstermesi için:

```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",

  "telephone": "+905356410304",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "Halkalı Merkez Mah. 1. Çiğdem Sok. No:12",
    "addressLocality": "Küçükçekmece",
    "addressRegion": "İstanbul",
```
