## 🧠 PSİKOLOJİK TEMELLERİ

### Hedef Profil
- **Yaş:** 18-45
- **Cinsiyet:** Kadın ağırlıklı (ama inclusive)
- **Sorun:** Wellness, self-care, success, günlük stres
- **Tech Savvy:** Ortalama - teknolojiden uzak

### Duygusal Ton (3 Kelime)
**Sıcak • Samimi • Yakın**

### Görsel Metafor
**"Sıcak Ev" (güvenli, samimi)**
- Kullanıcı eve geldiğinde rahatladığı gibi rahatlamalı
- Teknoloji değil, insan hissi vermeli
- Her gün aynı güvenli ortama dönüyor hissi

### Enerji Seviyesi
**7/10** — Dengeli (Ne çok sakin, ne çok enerjik)
- Headspace = 8 (çok enerjik, renkli)
- Calm = 3 (çok sakin, minimal)
- **Senle = 7** (sıcak, motive edici ama yorucu değil)

## 🔤 TİPOGRAFİ

### Logo Font
**Nunito Bold** (Alternatif: Comfortaa Bold)
- Rounded, dostane, sıcak
- **Lowercase:** "senle" (küçük harfle → daha samimi)
- Letter-spacing: +0.5px (hava alsın)
- Size: 48pt (logo)

### Display / Başlıklar (H1, H2)
**Manrope SemiBold** (Alternatif: Outfit Medium)
- Modern ama yumuşak
- Okunabilir (teknolojiden uzak kitle için önemli!)
- Size: 24-32pt (büyük, rahat)
- Letter-spacing: -0.5px (tight ama okunur)

### Body Text
**Inter Regular** (Alternatif: SF Pro Text Regular)
- En okunabilir fontlar
- Size: 16-18pt (standarttan büyük → yaşlı kullanıcılar için)
- Line-height: 1.6 (rahat okuma)
- Letter-spacing: normal

### Psikolojik Sebepler
- **Rounded ama abartısız:** Profesyonel + sıcak
- **Büyük fontlar:** Teknolojiden uzak kitle için erişilebilir
- **Sans-serif:** Modern ama intimidating değil
- **Lowercase logo:** Daha samimi, daha az "corporate"

### Tipografi Kuralları
- Max 3 heading seviyesi (H1, H2, Body)
- Font size scale: 16 / 20 / 24 / 32
- Bold kullanma → SemiBold yeterli (daha yumuşak)
- ALL CAPS kullanma → okunmaz, agresif

## 🖼️ GÖRSEL DİL

### İllüstrasyon Stili

#### ✅ KULLAN
- **Soft, organic şekiller** (blob shapes, yumuşak dalgalar)
- **Minimal line art** (kadın siluetleri, bitki motifleri, yoga pozları)
- **Warm gradient backgrounds** (cream→peach, beige→coral)
- **Rounded, flowing shapes** (keskin köşe yok)
- **2-3 renk max per illustration** (sade)

#### ❌ KULLANMA
- Headspace gibi cartoon karakterler (çok playful)
- Abstract geometric şekiller (soğuk, teknik)
- Keskin köşeler, hard edges
- Parlak, neon renkler
- Çok detaylı, karmaşık illustrasyonlar

### Fotoğraf Stili

#### ✅ KULLAN
- **Yumuşak doğal ışık** (sabah/akşam golden hour)
- **Ev içi ortamlar** (kanepe, yatak, pencere kenarı)
- **Detay çekimler:** Eller, bitki detayları, kupa, kitap
- **Kadınlar (diverse):** Farklı yaşlar, ten renkleri, saç stilleri
- **Minimal composition:** Az eleman, bol boşluk
- **Warm tones:** Cream, beige, terracotta dominant

#### ❌ KULLANMA
- Stock photo gülümsemeleri (fake hissi)
- Stüdyo çekimleri (çok corporate)
- Sadece genç beyaz kadınlar (inclusivity!)
- Soğuk renkler (mavi ışık, gri tonlar)
- Kalabalık kompozisyonlar

### İkonografi
- **Style:** Minimal, 2-3 stroke weight
- **Corners:** Rounded (keskin değil)
- **Shapes:** Organic (geometrik değil)
- **Fill:** Outline preferred (daha hafif)
- **Size:** 24x24px standard

## 🏠 UI TASARIM PRENSİPLERİ

### 1. Rounded Everything
```css
border-radius: 24px;  /* Kartlar, modals */
border-radius: 16px;  /* Butonlar */
border-radius: 12px;  /* Input fields */
border-radius: 8px;   /* Küçük elementler */
```
**Sebep:** Keskin köşeler = stres, yuvarlaklık = güven

### 2. Generous Spacing
```css
padding: 24-32px;     /* Kartlar */
margin: 16-24px;      /* Element arası */
gap: 16px;            /* Flexbox/Grid */
```
**Sebep:** Sıkışıklık = kaygı, boşluk = rahatlık

### 3. Soft Shadows
```css
box-shadow: 0 8px 24px rgba(74, 64, 58, 0.08);   /* Kartlar */
box-shadow: 0 4px 12px rgba(74, 64, 58, 0.06);   /* Butonlar */
box-shadow: 0 2px 8px rgba(74, 64, 58, 0.04);    /* Inputs */
```
**Sebep:** Floating hissi, hafiflik, depth

### 4. Minimal Hierarchy
- Max 3 heading seviyesi
- Her ekranda 1 ana aksiyon
- 2-3 buton max per screen
- Navigation simple (tab bar, 4-5 item max)

**Sebep:** Teknolojiden uzak kitle → basit, net olmalı

### 5. Gradients (Subtle)
```css
background: linear-gradient(135deg, #FFF8F0 0%, #FFE4D6 100%);
background: linear-gradient(180deg, #E8DCC8 0%, #A8C5DA 100%);
```
- **Kullan:** Backgrounds, hero sections
- **Kullanma:** Text, butonlar (okunmaz)
